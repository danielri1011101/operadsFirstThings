\section{}

According to Pierce, the operational view of $\forall X . T$ is that is values are functions
which map a type $S$ to a value of type $[X \mapsto S]T$. Namely,

\begin{eqnarray*}
	v : \forall X . T \ \vdash \ v \ S : [X \mapsto S]T.
\end{eqnarray*}
